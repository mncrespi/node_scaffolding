# Example Implementation OAuthServer v3

This Project is based on Node Scaffolding

URL: https://github.com/mncrespi/node_scaffolding 

## Grants Allowed: 
- password
- refresh_token
- client_credentials 

## Features

- ES6
- JWT and express-JWT
- esLint v4
- Nodemon
- debug
- bluebird
- JOI with express-validation
- CORS
- MongoDB
- oAuth2 Server v3
